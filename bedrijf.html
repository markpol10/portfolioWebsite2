Website Bedrijfkracht ontwikkelen
Tijdens het tweede jaar heb ik samen met een projectgroep een website mogen ontwikkelen voor een opdachtgever. Dit was in opdracht de Hanze, Enpuls en de Gasunie. Zij hebben het initiatief om mensen meer bewust te maken van hun eigen energieverbruik. Om dit te ondersteunen, vroegen zij ons om een website te ontwikkelen.

Wat heb ik gedaan
Voor dit project heb ik samen met een groep een volledig responsive website moeten ontwikkelen. Wj hebben ons gericht op het front-end gedeelte van de website. Hiervoor hebben wij een een framework gebruikt. Dit was het CSS-framework Bootstrap.

Wat heb ik geleerd
Wat ik voornamelijk heb geleerd tijdens dit project is dat ik stappen heb kunnen maken op gebied van HTML en CSS. Dit was ook de eerste keer dat ik echt tevreden was met het eindresultaat. Ook was dit mijn eerste ervaring met het gebruiken van een framework. Bootstrap is op dit moment nog steeds het framework dat ik het meeste gebruik.
