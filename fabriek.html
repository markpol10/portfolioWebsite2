Infographic en test om werkloosheid tegen te gaan
In de toekomst zullen veel banen verloren gaan doordat deze banen geautomatiseerd zullen worden. Samen met twee medestudenten hebben wij de opdracht gekregen om bewustwording hiervan te vergroten. Daarnaast hebben wij een beroepentest uitgewerkt dat er voor moet zorgen om alternatieve banen uit te lichten. Onze focus ging uit op banen binnen de transportsector.

Wat heb ik gedaan
Tijdens dit project heb ik de taak op mij genomen om mij voornamelijk te richten op het uitwerken van de infographic. Hierdoor heb ik mij kunnen ontwikkelen op mijn front-end development skills. Hoe dit er uit is komen te zien is te zien op het filmpje hiernaast. Daarnaast heb ik ook geholpen om de beroepentest op de Wordpress website te zetten.

Wat heb ik geleerd
Dankzij dit project heb stappen gemaakt in HTML, CSS en ik heb beginstappen gemaakt in Javascript. Ook heb ik tijdens het uitwerken van de test de basics van Wordpress onder de knie gekregen.
