Ontwerpen nieuwe mobiele KPN app
Het afgelopen schooljaar ben ik samen met een medestudent bezig geweest om een nieuwe mobiele app te ontwerpen in opdracht van KPN. De bedoeling was om KPN nieuwe inzichten te geven die zij kunnen verwerken in een mogelijke nieuwe app. Uiteindelijk hebben wij een app opgeleverd waarbij wij 3 nieuwe inzchten als basis hebben gebruikt.

Wat heb ik gedaan
Het doel van dit project was om een mobiele app te ontwikkelen voor KPN. Omdat mijn groep maar uit twee personen bestond, kon ik veel verschillende taken op mij nemen. De voornaaste taak waar ik mij mee bezig heb gehouden was het front-end development gedeelte van de app. Daarnaast was ik de projectleider gedurende het project.

Wat heb ik geleerd
Gedurende het project was er veel ruimte om onze eigen taken in te delen. Dit zorgde er voor dat er genoeg ruimte was voor persoonlijke ontwikkeling. Ten eerste heb ik er bewust voor gekozen om projectleider te zijn. Dit was een rol waar ik weinig ervaring mee had, maar ik kwam er dankzij dit project achter dat deze rol mij eigenlijk heel goed ligt. Ook heb ik mij kunnen ontwikkelen op gebied van front-end development, waar ik altijd veel plezier mee beleef.
